Data Structure Documentation

◆ Parameters

struct Parameters

Variable gathered for image calculation.

There are three sources for this data. The first is command line processing. This is by way of short/long options via getopt_long(). The command line may also be processed without options in fixed order. The second is the use of a configuration file that essentially sets the default values for the necessary parameters. The third method is parsing a JSON file which contains the values for a particular fractal image.

enum innerLoop

Choice values for inner loop calculation.

Currently 4 values in this list. COMPLEX is the default method. ZX uses all real values, no complex variables. DEMM is the Distance Estimator Method for the Mandelbrot set. MSETPOT is the Mandelbrot set via Continuous Potential Method.

Enumerator
COMPLEX 

ƒ(Z) = Z^2 + C where Z,C are complex

ZX 

ƒ(Z) = as above using real ZX, ZY, CX, CY

DEMM 

Uses Distance estimation method

MSETCPM 

Uses Continuous Potential method
