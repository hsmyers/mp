24 ### Getting Started/Requirements/Prerequisites/Dependencies
25 Include any essential instructions for:
26 - Getting it
27  * Engage the usual suspects; download or clone&hellip;
28 - Installing It
29  * Make sure you have a usable GCC
30  * I use msys2 on windows and that insures GCC 6.3.0
31  * you will need libs jpeg, gd and quadmath. openmp as well&hellip;
32  * compile using GCC;
33  * gcc -Wall -o mp -std=c99 -O3 -msse2 -march=native -fopenmp palette.c dictionary.c iniparser.c getopt.c cnames.c cspace.c colors.c elapsed.c util.c cJSON.c mp.c -ljpeg -lgd -lquadmath
