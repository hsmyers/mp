Detailed Description

This is the primary source file for mp.exe.

Author
Hugh S. Myers
Date
Mon Aug 14 10:25:06 2017

DESCRIPTION

This is both the location of main() as well as the central file for the executable. Mp.exe is an OpenMP Mandelbrot image generator. It is designed to be open ended as well as an on-going test bed. As such not everything may work. You've been warned. Best indicator of something wrong is an image that is all black. I suppose in keeping with the windows platform, the image should be all blue, but it is what it is.

Note
compile: gcc -Wall -o mp -std=c99 -O3 -msse2 -march=native -fopenmp cnames.c cspace.c colors.c elapsed.c util.c cJSON.c mp.c -ljpeg -lgd
gcc -ggdb -Wall -o mp -std=c99 -msse2 -march=native -fopenmp cnames.c cspace.c colors.c elapsed.c util.c cJSON.c mp.c -ljpeg -lgd
debug: gdb -args …
Bug:
No known bugs.

◆ MSetDBL()

Parameters MSetDBL ( Parameters  P)

Mandelbrot Set using double.

Function to compute the Mandelbrot Set using double variables. This derives an image g.width × g.height, centered at g.center from g.centerX and g.centerY. The complex width is g.diameter. The inner loop is available in 4 methods; ZX, meaning calculate without complex variables, COMPLEX, meaning calculate with complex variables, DEMM, meaning use the Distance Estimator Method for Mandelbrot set and MSETCPM, meaning use the Continuous Potential Method.

Parameters
PParameters structure, a collection of configuration information.
Returns
