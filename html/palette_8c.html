Detailed Description

A collection of functions in aid of RGB palettes.

Author
Hugh S. Myers
Date
Mon Jul 24 11:20:10 2017

This module provides the construction of dynamic palettes. Essentially a contiguous array of RGB values, which can be either loaded from a file or constructed dynamically.

No known bugs.
