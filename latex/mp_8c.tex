This is both the location of \hyperlink{mp_8c_a0ddf1224851353fc92bfbff6f499fa97}{main()} as well as the central file for the executable. Mp.\+exe is an Open\+MP Mandelbrot image generator. It is designed to be open ended as well as an on-\/going test bed. As such not everything may work. You\textquotesingle{}ve been warned. Best indicator of something wrong is an image that is all black. I suppose in keeping with the windows platform, the image should be all blue, but it is what it is.

Mandelbrot Set using long double. 

Function to compute the Mandelbrot Set using long double variables. This derives an image g.\+width × g.\+height, centered at g.\+center from g.\+centerX and g.\+centerY. The complex width is g.\+diameter. The inner loop is available in 4 methods; ZX, meaning calculate without complex variables, C\+O\+M\+P\+L\+EX, meaning calculate with complex variables, D\+E\+MM, meaning use the Distance Estimator Method for Mandelbrot set and M\+S\+E\+T\+C\+PM, meaning use the Continuous Potential Method.
