\subsubsection{struct Parameters}
Variable gathered for image calculation. 

There are three sources for this data. The first is command line processing. This is by way of short/long options via getopt\+\_\+long(). The command line may also be processed without options in fixed order. The second is the use of a configuration file that essentially sets the default values for the necessary parameters. The third method is parsing a J\+S\+ON file which contains the values for a particular fractal image.
